# Transaction Data Analysis Dashboard (Excel)

## Problem Statement

This project focuses on analyzing transaction data to generate meaningful business insights.  
The dashboard helps in understanding revenue, expenses, profit trends, and customer transaction behavior.  

It also enables comparison across different categories, regions, and payment methods, helping in better decision-making and performance tracking.

---

## Steps Followed

- Step 1 : Imported and cleaned the dataset in Microsoft Excel  
- Step 2 : Performed data transformation for structured analysis  
- Step 3 : Created Pivot Tables for summarizing transaction data  
- Step 4 : Designed an interactive dashboard layout  

- Step 5 : Created visuals:
  - Revenue, Expense, and Profit comparison by category  
  - Profit analysis based on payment methods  
  - Revenue distribution across product lines  
  - Region-wise transaction analysis  
  - Average expense visualization using pie chart  

- Step 6 : Added slicers/filters for dynamic data interaction  

- Step 7 : Organized dashboard for clear and user-friendly insights
